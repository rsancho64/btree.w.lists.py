## btree.w.lists.py

Una libreria para representar arboles binarios como listas.

```bash
*vacio* [none,none,none]

22 : [22,none,none]

  22
11    [22,[11,none,none],none]

  22
11  33    [22,[11,none,none],[33,none,none]]

  22
    33    [22,none,[33,none,none]]

*BAD* : [22,[33,none,none]]
```
